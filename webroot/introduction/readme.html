coolie 官方指南

介绍

coolie 主张最小、最方便的宗旨，和最小的侵入原则。不会改变前端开发者的编码习惯，不会大量侵入开发环境下的代码， 不必在开发过程中监听构建。

  • 如果没有 coolie，你的项目仍然可以运行。
  • 如果加上 coolie，那么你的项目将健步如飞。

coolie 主要服务于生产环境的代码，将代码进行修身减肥，却不失一些美的特征。 使你的代码在生产环境、用户的浏览器、移动手机上跑的更快，使你获得更多的赞赏和表扬。 为你当上总经理，出任 CEO，迎娶白富美，走向人生巅峰奠定最基础的一步。

是不是觉得 coolie 还是有那么点意思？那么，现在开始了解它吧。
