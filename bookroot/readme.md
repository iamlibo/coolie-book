# coolie
coolie —— 可能是你心目中最理想的前端开发构建工具。

- JS 文件的分析、合并、压缩、版本管理
- CSS 文件的分析、合并、压缩、版本管理
- HTML 文件分析、压缩、版本管理
- 入口模块的分析、分块、合并、压缩、版本管理
- 静态资源的分析、压缩、版本管理
